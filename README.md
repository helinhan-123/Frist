# 企业网络 eNSP 实验
基于华为 eNSP 模拟器的企业网络拓扑与配置实验，用于学习和复现典型企业网架构。

## 📁 项目结构

## ✨ 实验目标
- 搭建符合企业场景的分层网络架构（核心层、汇聚层、接入层）
- 完成 VLAN、OSPF、ACL、NAT 等典型网络技术配置
- 验证企业网内外部通信、安全策略与路由可达性

## 🛠️ 环境要求
- 华为 eNSP 模拟器（推荐版本：V100R003C00 及以上）
- 本地计算机满足 eNSP 运行硬件要求

## 🚀 使用方法
1. 下载本仓库的 `企业网络ensp实验.topo` 文件
2. 打开 eNSP 模拟器，点击「文件 → 打开拓扑」
3. 选择下载的 `.topo` 文件，加载实验拓扑
4. 依次启动设备，按实验文档完成配置与验证

## 📌 实验内容（示例）
- 设备：路由器 AR2220、交换机 S5700、云设备（模拟公网）
- 技术点：
  - OSPF 动态路由协议部署
  - 访问控制列表 ACL 实现流量过滤
  - NAT 地址转换实现内网访问公网

## 📄 备注
- 拓扑文件仅包含设备与链路连接，具体配置需手动完成
- 可在本仓库补充配置脚本与实验报告
